# 사용자 가이드 (User Guide)

News Curation 서비스를 사용하는 방법을 설명합니다.

## 🏠 메인 화면 (News Feed)

서비스에 접속(`http://localhost:3000`)하면 최신 뉴스 피드를 확인할 수 있습니다.

### 주요 기능
1. **카테고리 탭 (Category Tabs)**
   - 상단의 탭을 클릭하여 '기술', '과학' 등 관심 있는 분야의 뉴스를 필터링해서 볼 수 있습니다.
   - '전체' 탭에서는 모든 카테고리의 최신 뉴스를 시간순으로 보여줍니다.

2. **뉴스 카드 (News Card)**
   - 각 뉴스는 카드 형태로 표시되며, 썸네일, 제목, 출처, 발행일시가 표시됩니다.
   - **AI 요약**: 카드에는 OpenAI가 생성한 3줄 요약이 포함되어 있어 기사 내용을 빠르게 파악할 수 있습니다.
   - 카드를 클릭하면 원본 기사 페이지로 이동합니다.

3. **무한 스크롤 (Infinite Scroll)**
   - 페이지 하단으로 스크롤하면 자동으로 이전 뉴스를 추가로 불러옵니다.

## 🛠️ 관리자 페이지 (Admin Page)

관리자 기능은 `/admin` 경로로 접속하여 이용할 수 있습니다. (예: `http://localhost:3000/admin`)

### 1. 카테고리 관리
- 새로운 뉴스 카테고리를 추가할 수 있습니다.
- 카테고리 이름과 뉴스 수집 시 사용할 검색어(Query)를 입력하여 등록합니다.

### 2. 수집 설정 (Collection Settings)
- **자동 수집 스케줄**: 뉴스를 자동으로 수집할 주기를 설정할 수 있습니다 (Cron 표현식 사용).
- **수동 수집 실행**: '지금 수집하기' 버튼을 눌러 즉시 뉴스를 수집할 수 있습니다.

### 3. 로그 확인
- 뉴스 수집 로그, API 호출 로그 등을 확인하여 시스템 상태를 모니터링할 수 있습니다.
